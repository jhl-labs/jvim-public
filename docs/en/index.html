jvim

v0.5.3 Linux x64 · arm64 single binary

Notepad-ergonomic, vim-fast, AI-native.

Terminal markdown editor for notes. wikilinks · tags · git · semantic search · AI overlay in a single binary. 1-line install, npm recommended.

Get started in 5 min Install
npm install -g @jhl_labs/jvim

Full video: jvim_demo.mp4

Features

md

Markdown-first editing

Live styling for headings, emphasis, lists, code fences, tables, inline links. Workspace, not pages.

[[ ]]

Wikilinks + Backlinks

Type [[ to insert. Backlink panel shows incoming references.

#

Tag browser

#tag autocomplete + F7 tag browser. Vault-wide frequency and top files.

find

Full-text + semantic search

Shift+F4 vault search, Shift+F5 index scope, RAG sources in Ask mode. SQLite FTS5 + optional sqlite-vec.

cmd

Notepad shortcuts

Ctrl+C/X/V/A/Z/Y/S/O/N + Shift+Arrow selection. Plus vim-style tree (j/k/g/G/n/r/d).

AI

AI overlay

Ask/Insert/Rewrite, inline autocomplete, document polish, RAG sources. Local LLM supported.

git

Git integration

Diff panel, current file hunks, commit/push/pull without leaving the editor.

bin

Single binary

Bun-compiled. No node_modules, no Node runtime required.

Why jvim?

  • No mode switching. Insert mode is the default — like Notepad. vim users keep j/k/g/G/n/r/d in the tree.
  • Markdown vault, not pages. Open a folder, navigate with the file tree, link with [[wikilinks]], tag with #topics.
  • AI that knows your notes. Ask mode searches your vault first (RAG), then composes — citing the sources.
  • Single binary. Download, run. Linux x64 / arm64.
