# Asciinema casts

각 워크플로우 녹화 파일.

녹화 방법:
1. `asciinema rec -i 2 --title "<title>" editor-basics.cast`
2. 30~60초 안에 핵심 동작 시연
3. Ctrl+D로 종료
4. 이 디렉터리에 커밋

권장 cast 목록:
- editor-basics.cast — 파일 열기 · 편집 · 저장
- file-tree.cast — 트리 이동 · 필터 · n/r/d
- vault-search.cast — Shift+F4 검색
- ai-overlay.cast — Ask · Insert · Rewrite
- git.cast — diff 패널 · current-file hunks
