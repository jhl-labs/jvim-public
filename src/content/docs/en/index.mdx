## Why jvim?

- **No mode switching.** Insert mode is the default — like Notepad. vim users keep `j/k/g/G/n/r/d` in the tree.
- **Markdown vault, not pages.** Open a folder, navigate with the file tree, link with `[[wikilinks]]`, tag with `#topics`.
- **AI that knows your notes.** Ask mode searches your vault first (RAG), then composes — citing the sources.
- **Single binary.** Download, run. Linux x64 / arm64.
