## 왜 jvim인가?

- **모드 전환 없음.** 기본이 삽입 모드 — Notepad처럼. vim 사용자는 파일 트리에서 `j/k/g/G/n/r/d`를 그대로 사용.
- **마크다운 vault, 페이지 아님.** 폴더를 열고, 파일 트리로 탐색하고, `[[wikilinks]]`로 연결하고, `#주제`로 태그.
- **내 노트를 아는 AI.** Ask 모드는 vault를 먼저 검색(RAG)한 후 출처를 인용해서 답변.
- **단일 바이너리.** 다운로드하고 실행. Linux x64 / arm64.
