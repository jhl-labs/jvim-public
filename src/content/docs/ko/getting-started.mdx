이 가이드를 따라가면 5분 안에 vault가 완성됩니다.

## 1. 설치 (1분)

```bash
npm install -g @jhl_labs/jvim
jvim --version    # 기대 출력: jvim 0.5.x
```

`command not found`가 뜨면 [설치 트러블슈팅](/jvim-public/ko/install/#troubleshooting)을 확인하세요.

## 2. 첫 vault 만들기 (30초)

```bash
mkdir my-notes
jvim my-notes/
```

왼쪽에 파일 트리(비어 있음), 오른쪽에 에디터가 열립니다.

## 3. 첫 노트 작성하기 (1분)

에디터에서 `Ctrl+N`을 눌러 새 파일을 만드세요. 파일 이름을 입력합니다: `welcome.md`. 그런 다음 작성:

```markdown
# Welcome

This is my first jvim note.

- [[ideas]] for things to write later
- #journal #setup
```

`Ctrl+S`를 누르세요. 상태 표시줄의 `●` 마커가 사라지면 저장된 것입니다.

## 4. 탐색과 검색 (1분)

- `Ctrl+E` — 파일 트리로 포커스 이동. `j`/`k`로 이동, `Enter`로 열기.
- `Ctrl+O` — vault 전체 퍼지 파일 검색.
- `Shift+F4` — 전체 노트 검색 (전문 검색 + 태그 + wikilinks).
- `F2` — 현재 문서의 아웃라인 (제목 점프).

## 5. AI에게 묻기 (1분)

`Alt+A`를 누르면 Ask 패널이 열립니다. 질문을 입력하면 — vault를 먼저 검색한 후 출처 노트를 인용해서 답변을 작성합니다.

> AI 프로바이더가 설정되어 있어야 합니다. 기본적으로 jvim이 첫 사용 시 안내합니다.
> 프로바이더 설정(OpenAI / Anthropic / 로컬)은 [AI 오버레이](/jvim-public/ko/usage/ai-overlay/)를 참고하세요.
